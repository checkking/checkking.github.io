https简单过程

  • 客户端请求服务器拿到公钥，以及签名
  • 客户端验证服务器下发的公钥是否有合法(通过签名)
  • 客户端随机生成密钥，并通过公钥加密发送给服务端
  • 客户端和服务端通过密钥进行数据传输
